# E 系列环境素材生产清单（给 Codex）

沙海群洲世界的环境道具。风格基准与 A01-A10 一致：方块体素、硬边低模、像素化色块。
提示词 ≤150 字（混元网页版限制）。产出 GLB 放 `assets/hunyuan/raw/`，命名 `E0X_用途_v1.glb`。

## 硬性验收标准（任一不满足即打回重生成）

1. **单一主体，无底座、无地面、无小岛托盘**——游戏里有真实地形，带底座必穿模。提示词末尾必须带"无底座无地面"。
2. 独立模型，一个 GLB 一个物件，不要打包场景。
3. 方块/体素构造，硬边，无平滑曲面、无写实材质。
4. 3/4 视角轮廓清晰可辨认。
