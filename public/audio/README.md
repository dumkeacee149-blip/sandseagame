# Audio Sources

These audio files are selected from Kenney audio packs and kept under stable project names.

- `music/sandsea-loop.ogg`: Kenney Sound Pack, Music Loops, `Flowing Rocks.ogg`, CC0.
- `sfx/ui-*.ogg`: Kenney Interface Sounds, CC0.
- `sfx/coins.ogg`, `sfx/dock.ogg`, `sfx/board.ogg`, `sfx/equip.ogg`, `sfx/harpoon.ogg`, `sfx/sword.ogg`: Kenney RPG Audio, CC0.
- `sfx/wood-break.ogg`, `sfx/metal-hit.ogg`, `sfx/hurt.ogg`, `sfx/sand-step.ogg`: Kenney Impact Sounds, CC0.
- `sfx/treasure.ogg`, `sfx/victory.ogg`: Kenney Music Jingles, CC0.

Kenney credits are appreciated but not required by the license.
